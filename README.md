# SSH KEY COPY
ПЗ для копіювання публічних ключів на Unix сервери

# Як зібрати самому
1. Скопіювати проєкт
2. Перенести на любий доступний сервер Linux Unix
3. Для Debian дистрибутивів
  - apt-get install mingw-w64 imagemagick osslsigncode openssl nsis
4. make icons (створення файла *.ico)
5. make windows (створення *.exe)
6. make installer (створення інсталятора)
